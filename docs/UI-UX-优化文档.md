# Wealthy UI/UX 重构文档

## 🎨 设计系统优化

### 主题配色方案
- **主色调**：优雅的深蓝绿色 (hsl(195, 85%, 41%))
- **背景色**：温暖的浅色系统 (hsl(0, 0%, 99%))
- **深色模式**：现代优雅的深色配色
- **状态色**：
  - 成功 (Success): hsl(142, 76%, 36%)
  - 警告 (Warning): hsl(38, 92%, 50%)
  - 错误 (Destructive): hsl(0, 84%, 60%)
  - 信息 (Info): hsl(217, 91%, 60%)

### 圆角与间距
- 基础圆角：0.75rem (12px)
- 卡片圆角：1rem - 2rem
- 统一的内边距系统：4px, 8px, 16px, 24px, 32px

### 阴影系统
- shadow-sm: 轻微阴影，用于悬停提示
- shadow: 标准阴影，用于卡片
- shadow-md: 中等阴影，用于弹窗
- shadow-lg: 大阴影，用于模态框
- shadow-xl: 超大阴影，用于重要元素

## 🧩 新增UI组件库

### Card.vue
可复用的卡片组件，支持多种变体：
- `default`: 默认卡片样式
- `gradient`: 渐变背景卡片
- `glass`: 玻璃态效果卡片
- `bordered`: 边框加粗卡片

### Badge.vue
徽章组件，用于状态标识：
- 多种颜色变体（primary, success, warning, destructive, info）
- 三种尺寸（sm, md, lg）

### StatCard.vue
统计卡片组件：
- 展示标题、数值、描述
- 支持图标显示
- 支持趋势指示器
- 多种颜色变体

### EmptyState.vue
空状态组件：
- 友好的空状态提示
- 支持自定义图标和操作按钮
- 统一的空状态设计语言

### Dialog.vue
对话框组件：
- 现代化的模态框设计
- 支持自定义头部和底部
- 平滑的进出动画
- 响应式尺寸控制

### 增强的Button组件
- 新增变体：primary, success
- 支持加载状态
- 支持左右图标
- 更大的尺寸选项（xl）
- 优化的悬停效果和动画

### 增强的Input组件
- 支持左右图标
- 错误状态显示
- 更好的焦点样式
- 圆角优化

## 🎬 动画与过渡

### 全局动画类
- `.animate-in`: 淡入并上移动画
- `.animate-out`: 淡出动画
- `.transition-smooth`: 平滑过渡（0.2s cubic-bezier）

### 特殊效果
- 列表项延迟动画（stagger animation）
- 按钮悬停缩放效果
- 图标旋转动画
- 卡片悬停阴影增强

### 自定义滚动条
- 细窄设计（8px宽度）
- 圆角滑块
- 悬停状态变化
- 透明轨道

## 🚀 使用建议

### 导入UI组件
```vue
<script setup lang="ts">
import { Button, Card, Badge, StatCard, EmptyState } from '@/components/ui';
</script>
```

### 使用统一的工具类
```vue
<div class="glass shadow-smooth-lg transition-smooth">
  <!-- 玻璃态效果 + 大阴影 + 平滑过渡 -->
</div>
```

### 保持设计一致
- 使用设计系统中定义的颜色
- 遵循组件的使用规范
- 保持动画时长一致（0.2s）
